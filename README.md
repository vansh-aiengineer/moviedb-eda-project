 # MovieDB Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a real-world MovieDB dataset containing information about movies, ratings, popularity, genres, languages, and release dates.

The goal is to discover trends, patterns, and insights from movie data using Python data science libraries.

## Dataset Information

The dataset contains the following features:

* Release_Date
* Title
* Overview
* Popularity
* Vote_Count
* Vote_Average
* Original_Language
* Genre
* Poster_Url

Total Records: 9,837 Movies

## Objectives

* Clean and preprocess the dataset
* Analyze movie popularity and ratings
* Explore genre and language distributions
* Identify top-rated and most popular movies
* Visualize trends using charts and graphs

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

### Data Cleaning

* Removed duplicate records
* Converted date columns
* Handled invalid values

### Exploratory Data Analysis

* Top 10 Most Popular Movies
* Movies Released Per Year
* Top Languages
* Top Genres
* Ratings Distribution
* Popularity vs Vote Average
* Correlation Analysis

## Key Insights

* Identified the most popular movies based on popularity scores.
* Analyzed the most common movie genres.
* Explored language distribution across movies.
* Studied relationships between popularity, ratings, and vote counts.
